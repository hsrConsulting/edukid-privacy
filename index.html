Politique de confidentialité — EduKid

Application éducative pour enfants âgés de 6 à 10 ans (CP à 6e)

Éditeur : HSR Consulting
Application : EduKid (Android, com.hsrconsulting.edukid.app)
Contact : hsr.consulting.pro@gmail.com
Dernière mise à jour : 12 mai 2026

EduKid est une application éducative destinée aux enfants âgés de 6 à 10 ans (classes CP à 6e). Nous prenons la protection des données des enfants très au sérieux et respectons le Règlement Général sur la Protection des Données (RGPD), notamment ses dispositions renforcées pour les mineurs (RGPD-K / article 8), ainsi que les exigences COPPA pour les utilisateurs résidant aux États-Unis.

1. Données que nous collectons

1.1 Renseignées par l'enfant ou son parent au démarrage

  • Prénom (ou pseudonyme) de l'enfant
  • Niveau scolaire (CP, CE1, CE2, CM1, CM2, 6e)
  • Nom de l'établissement scolaire et éventuel nom de classe
  • Avatar emoji (choisi parmi une liste)
⚠️ Recommandation : nous recommandons fortement d'utiliser un pseudonyme plutôt que le prénom réel, et de ne pas saisir d'informations permettant d'identifier précisément l'enfant.

1.2 Générées par l'usage de l'application

  • Réponses aux exercices (correct / incorrect, temps passé, difficulté)
  • Points, badges, niveau de progression
  • Historique de chat avec l'assistant pédagogique (uniquement si la fonctionnalité « EduBot » est activée — voir section 4)

1.3 Techniques (anonymes)

  • Identifiant d'installation Firebase (anonyme, régénérable)
  • Données de plantage techniques (Firebase Crashlytics) — sans contenu personnel
  • Statistiques d'usage agrégées (Firebase Analytics)

3. Publicité et achat « Premium »

EduKid affiche une petite bannière publicitaire sur les écrans de menu (Accueil, Progrès, Profil). Ces publicités respectent strictement les règles « Designed for Families » de Google Play :

  • Aucune publicité personnalisée. Aucun identifiant publicitaire (AAID) n'est utilisé, aucun ciblage comportemental.
  • Les annonces proviennent uniquement de réseaux Families Self-Certified approuvés par Google pour les apps enfants.
  • Configuration technique : tagForChildDirectedTreatment = TRUE et maxAdContentRating = G (tout public).
  • Aucune publicité n'apparaît pendant les exercices, les quiz, les dictées ou les défis — uniquement sur les écrans de navigation.

Acheter EduKid Premium (achat unique via Google Play) désactive définitivement toute publicité dans l'application. Aucune donnée personnelle supplémentaire n'est collectée — Google Play gère le paiement et nous communique seulement le statut « acheté/non acheté ».

4. Données que nous NE collectons PAS

  • Adresse e-mail de l'enfant
  • Numéro de téléphone
  • Adresse postale précise
  • Géolocalisation
  • Photos ou vidéos
  • Carnet de contacts
  • Données biométriques

4. Fonctionnalité « EduBot » (assistant IA)

L'application propose un assistant pédagogique optionnel basé sur le modèle Claude (fourni par Anthropic). Lorsque l'enfant pose une question :

  • Le texte de la question est envoyé à l'API Anthropic via une connexion chiffrée HTTPS pour générer une réponse adaptée.
  • Aucune donnée personnelle (prénom, classe, école) n'est transmise.
  • Anthropic conserve les requêtes selon sa propre politique (anthropic.com/legal/privacy).
  • Cette fonctionnalité est verrouillée par défaut derrière le code PIN parent (4 chiffres) et peut être désactivée dans les paramètres.

5. Tableau de bord enseignant (web)

Si l'enseignant souhaite suivre la progression de sa classe :

  • Il doit créer un compte sur notre tableau de bord web (email + mot de passe).
  • Il accède uniquement à des statistiques agrégées et au classement des élèves de sa classe (par prénom/pseudonyme).
  • Aucune information personnelle des enfants au-delà du prénom n'est accessible.

6. Partage avec des tiers

Nous ne vendons pas les données. Les seuls tiers ayant accès à certaines données techniques sont :

7. Droits des parents / représentants légaux

Conformément au RGPD, vous avez les droits suivants :

  • Droit d'accès : connaître les données stockées sur votre enfant
  • Droit de rectification : corriger des informations inexactes
  • Droit à l'effacement (« droit à l'oubli ») : supprimer toutes les données — c'est immédiat via « Réinitialiser le profil » dans les paramètres (PIN parent requis), ou en désinstallant l'application
  • Droit à la portabilité : recevoir une copie des données
  • Droit d'opposition : refuser certains traitements (ex. Analytics)

Pour exercer ces droits, contactez : hsr.consulting.pro@gmail.com
Vous pouvez aussi déposer une réclamation auprès de la CNIL.

8. Sécurité

  • Toutes les communications avec nos serveurs utilisent HTTPS/TLS.
  • Les données stockées sur le téléphone sont protégées par les mécanismes natifs Android (chiffrement de l'appareil par défaut depuis Android 7).
  • Aucun mot de passe enfant n'existe (authentification anonyme uniquement).
  • Le code PIN parent (4 chiffres) reste sur le téléphone, jamais transmis.

9. Enfants de moins de 13 ans

Conformément à COPPA (USA) et à l'article 8 du RGPD :

  • Aucune publicité personnalisée
  • Aucun achat in-app réel (les « points » et « badges » sont virtuels)
  • Pas de chat libre avec des inconnus (les défis sont limités aux camarades de classe identifiés par l'enseignant)

Nous recommandons aux parents de configurer l'application avec leur enfant et de définir un code PIN parent pour verrouiller les paramètres sensibles.

10. Modifications de cette politique

Si nous modifions cette politique, nous publierons la nouvelle version dans l'application et sur la page web associée, en indiquant clairement la date de mise à jour. Pour les changements substantiels affectant la vie privée des enfants, nous solliciterons un nouveau consentement.
